# SimpleStampCalendar

世界一シンプルな学習記録アプリを目指しています。

<img width="441" height="409" alt="image" src="https://github.com/user-attachments/assets/b9dae412-c3ca-4cc3-a0d9-9601151f5242" />

カレンダーの日付をタップすると ⭐️ が付きます。もう一度タップすると消えます。データはブラウザの LocalStorage に保存しているため、サーバーやデータベースは不要です。

### 使い方

1. HTMLファイルを iCloud Drive や Google Drive に保存する
2. Edge で HTML ファイルを開く （スマホのChromeやSafariだとローカルのHTMLファイルを開けないので）
3. 学習した日に日付をタップする

以上です。
